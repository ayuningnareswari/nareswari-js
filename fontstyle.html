How to install fonts:
1) Select the font you want
2) Copy the code
3) Go to your customize page and paste the code between <style type="text/css"> and </style>
4) In the css code put
font-family: NAME OF FONT;
how to install fonts
How to upload your own fonts:
1. Go here & upload the font from your font file
2. Press "upload file" (Do not close tab yet)
3. Go to your customize page
4. Copy the following code below and paste it between <style type="text/css"> and </style>
@font-face {font-family: FONT NAME HERE; src:url(FONT URL HERE;)}
5. Go back to previous tab where you uploaded the font
6. Copy the link they gave you when you uploaded the file
7. Paste the link in the code where it says FONT URL HERE
8. Replace FONT NAME HERE with the name of the font
9. In the css code put
font-family: NAME OF FONT;
(Replace FONT NAME HERE with the name of the font you used before)

How to download fonts:
1) Press the install/download button on page
2) Unzip file if needed
Step 3)
  • Windows 7/Vista: Right-click on the font files > "Install"
  • Windows XP: Put the font files into C:\Windows\Fonts
  • Mac OS X: Double-click the font file > "Install font" button
